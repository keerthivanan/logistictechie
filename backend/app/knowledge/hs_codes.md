# HS Codes Guide for Common Cargo

## What is an HS Code?
Harmonized System (HS) codes are standardized numerical codes used worldwide to classify traded products. They determine customs duty rates and import regulations.

## HS Code Structure
- 6 digits internationally standardized
- Countries add digits for local classification
- Example: 8471.30 = Portable computers (laptops)

## Duty Rates in Saudi Arabia by Category
- Electronics: 0-5%
- Machinery: 0-5%
- Vehicles: 5%
- Textiles: 5-12%
- Food: 0-15% (varies)
- Chemicals: 0-5%

## Tips
- Always declare correct HS code — misdeclaration = fines
- Saudi Arabia uses 8-digit HS codes
- UAE uses 8-digit HS codes
- When unsure, consult a customs broker
- Use WCO database or customs authority website to verify
